Comment

A comment is a programmer-readable annotation in the source code of the computer program. Comments are used to make source code of the program readable & helps in documenting the code. Comments in JavaScript program are ignored by the JavaScript interpreter.

In JS, there are two types of comments :

1. Single line comment:

A single line comment begins with // & ends at the end-of-line. Here's how it looks like :

//This is a single line comment.
//Single line comments end with EOL.

2. Multi-line comment :

A Multi-line comment begins with /* & ends with */. Here's how it looks like :

/*This is a
multi-line comment.*/

Variables

A variable is the name of the memory location, where the data is stored. In terms of, from where the variable can be accessed, it can be classified into two types :

1. Local variable: When a variable's value is accessible only within a particular block of code (e.g. function). Thus it's scope is local.

2. Global variable: When a variable's value is accessible everywhere within the program. Thus it's scope is global.

Rules for naming a variable :

  1. 1 Name of the variable can contain characters (a-z)/(A-Z), numbers (0-9), dollar symbol ($) & underscore (_).
  2. 2 Name of the variable cannot start from a number (0-9).
  3. 3 Name of the variable is case sensitive.

Case sensitivity in JS : Since, JS is case sensitive, so variables with same name, but different casing wouldn't be considered as same.

var varName;
//is not same as
var VARNAME;
//is not same as
var vArNaMe;

Syntax for declaration of a variable :

var nameOfVariable;

Initialization of a variable refers to assignment of an initial value to a variable, since uninitialized variable's value by default is undefined.

Syntax for declaration & initialization of a variable :

var nameOfVariable = 15;

The equals to symbol (=) in the above syntax is not the mathematical equals to operator, rather it is the assignment operator, which assigns value on the RHS of the operator to the variable on the LHS of the operator.

Primary data types which can be stored in a variable :

  1. 1 Number
  2. 2 String
  3. 3 Boolean
  4. 4 Object
  5. 5 Undefined
  6. 6 Null

JavaScript is a dynamically typed language (like Python, Ruby), which means that same variable can be assigned values of different data types.

var var1 = 123; //var1 has stored a number
var1 = "asxyzp"; //var1 has stored a string
var1 = {first_name : "John",last_name : "Doe"}; //var1 has stored an object
var1 = true; //var1 has stored an Boolean

Number :

JS only has one type of number data type, unlike other programming languages such as C/C++, which has data types such as int, float, double & more. Numbers can be written with or without decimal point. Extra large or extra small numbers can be written with (exponential) scientific notation.

var number1 = 123 ; //Integer
var number2 = 123.0 ; //Floating point number
var number3 = 123e1 ; //=1230 (exponential notaion)
var number4 = 123e-1 ; //=12.3 (exponential notaion)

String :

String is a series of characters within single quotes 'STRING' or double quotes "STRING". You can use single quotes inside double quotes or double quotes inside single quotes, but using single quotes within single quotes or using double quotes within double quotes will give syntactical error.

var string1 = "'Hello' World" ; //Output = 'Hello' World
var string2 = '"Hello" World' ; //Output = "Hello" World

Adding a number with a string :

var numStr1 = 10 + 12 + " volvos"; //Output = '22 volvos' (string)
var numStr2 = "volvo no." + 10; + 12; //Output = 'volvo no. 1012' (string)

In the first case, addition of 10 with 12 happens before the sum is implicitly converted & then concatenated to " volvos", while in the second case, 10 & 12 are directly concated at the end of "volvo no. " without any implicit conversion.

Boolean :

A boolean variable can only have two values : true or false. Booleans are used for condition testing.

console. log( 1 = 1)  //output = true
console. log( 1 = 2)  //output = false

Object :

An object is a collection of properties, where a property is an association between a name (key) & a value. An object contains a name-value pairs, which are separated by comma (,).

var personObject = {first_name: "Jon",last_name: "Doe",age: 24,location: "Dehradun, India"};

Undefined :

In JavaScript, a variable which hasn't been initialized, has the default value of undefined. Also, any variable can be emptied by assigning it the value undefined. undefined represents unintentional abscence of object value.

var varName;
console. log( varName);  //output = undefined
var x = 1;
x = undefined;
console. log( varName);  //output = undefined

Null :

In JavaScript, null represents intentional abscence of value (unlike undefined, which represents unintentional abscence of value). Data type of null is object.

var varName = null;
console. log( varName);  //output = null

String

String is a sequence of characters, within single quotes or double quotes. A string can contain zero or more characters.

Synatx of a string :
var strName = "C1C2....CN";  //A string with N characters where Ci is a single character

Escaping literal quotes with escape sequence :

Q: In the variables section, we had learnt that we can't use "" inside "" or use '' inside '', but if we wanted to use quotes of a single type within another quote, then what should be done to achieve the same?

A: We can escape a "/' inside a "/', by using escape character (or a backlash \) to achive the same.

Here's how you can do it :

console. log( "\"Hello\", world!");  //Output = "Hello", world!
console. log( '\'Hello\', world!');  //Output = 'Hello', world!


Two reasons to use escape characters :

  1. 1 Escape characters allows to use characters, which may not otherwise be able to be typed out (e.g. \r - Carraige return).
  2. 2 Using quotes within quotes.

Different escape characters :

/' //Single quote
/" //Double quote
// //Backlash
/n //New line
/t //Horizontal tab
/v //Vertical tab
/r //Carraige return
/f //Formfeed
/b //Word boundary

String concatenation :

var str1= "STRING1", str2= "STRING2", str3;
str3= str1+ str2;  //Output (for str3) = "STRING1STRING2" (String concatenation) str1+= str2;  //Output (for str1) = "STRING1STRING2" (String concatenation with compound assignment with augmented addition)

Finding the length of a string :

str2. length;  //OUTPUT = 6, as length of "STRING2" is 6

Using [] notation to find characters in a string using index number :

Bracket notation ([]) is a way to get a character at a specific index in a string. The index starts from 0 & it represents the first character & this is why it's known as zero-based indexing.

var firstName= "asxyzp";
console. log( firstName[0]);  //OUTPUT = a
console. log( firstName[1]);  //OUTPUT = s
console. log( firstName[2]);  //OUTPUT = x
console. log( firstName[3]);  //OUTPUT = y
console. log( firstName[4]);  //OUTPUT = z
console. log( firstName[5]);  //OUTPUT = p

//How to find last character of a string
console. log( firstName [firstName .length- 1]);  //OUTPUT = p

//How to find Nth to last character of a string
console. log( firstName [firstName .length- N]);

String immutability :

Strings in JS are immutable, meaning that they can't be altered by changing individual values of characters by assigning them a value by uisng index. Check this out :

var firstName = "ASXYZP";
firstName[0] = "L";
console. log( firstName);  //Output = ASXYZP (No change because of string immutability)

Array

An array is a type of data structure, which holds multiple values in a single variable. Unlike C/C++/Java, where array of a particular type can only hold data of that type, array in JavaScript, can hold data of multiple types.

Syntax of declaration of an array :

var arrName1 =[ I1, I2,..., IN];  //Ik represents kth item in the array, which could be data of any type (number, string, boolean, object, ..).

var userDetails =[ 12563622262, "John", "Doe", "DED"];  //An array with a name 'userDetails' is storing the userID, first name, last name & location of the user in an array

Example of declaration of multi-dimensional array by nesting one array into another array :

var studentDetails =[ [["John","Doe"],73737, "Engineering"], [["Mogola","Roy Choudhury"],74467, "Arts"] ]; //A multi-dimensional array, which stores an array containing the first name & last name, studentID & stream

Accessing array elements with index no. :

Similar to a string, we can access individual elements of an array using bracket notation ([]) & an index number. Also, similar to a string, array uses a zero-based indexing.

Example of accessing array data using [] & index no. :

//How to access elements from a single dimensional array :
var numArr=[ 10, 20, 30];
console. log( numArr[ 0];  //Output = 10
console. log( numArr[ 1];  //Output = 20
console. log( numArr[ 2];  //Output = 30

//How to access elements from a multi dimensional array :
var studentDetails =[ [["John","Doe"],73737, "Engineering"], [["Mogola","Roy Choudhury"],74467, "Arts"]];
console. log( numArr[ 0][ 0][ 0];  //Output = "John"
console. log( numArr[ 0][ 0][ 1];  //Output = "Doe"
console. log( numArr[ 0][ 1];  //Output = 73737
console. log( numArr[ 0][ 2];  //Output = "Engineering"

Modifying array elements with index no. :

Unlike strings which are an immutable data type, meaning that they can't be modified by changing characters by using bracket notation ([]) with an index number, array elements can be modified by using the same. Here's an example of how the same can be achieved :

//How to modify elements from a single dimensional array :
var numArr=[ 10, 20, 30];
numArr[ 1] = 40;  //Output (of numArr) = [10,40,30]

//Process of changing the changing the element of a multi-dimensional array is the same as the process of changing the element of a single-dimensional array.

push() :

push() is a function, which appends data at the end of the array.

Syntax of push() function :

var arrName=[ I1, I2,..., IN];
arrName. push( IN+1);  //IN+1 is the value which will be pushed at the end of the array, making the array arrName = [I1,...,IN+1]

pop() :

pop() is a function, which removes array element from the end of the array & returning the value, which has been removed.

Syntax of pop() function :

var arrName=[ I1, I2,..., IN];
var valueReturned = arrName. pop();  //IN is the value which will be poped out of the array, making the array arrName = [I1,...,IN-1]

shift() :

shift() is a function, which removes array element from the beginning of the array, unlike pop(), which remove array element from the end of the array.

Syntax of shift() function :

var arrName=[ I1, I2,..., IN];
var valueReturned = arrName. shift();  //I1 is the value which will be shifted out of the array, making the array arrName = [I2,...,IN]

unshift() :

unshift() is a function, which adds an array element at the beginning of the array, unlike push(), which adds array element at the end of the array.

Syntax of unshift() function :

var arrName=[ I1, I2,..., IN];
arrName. shift(IN+1);  //I1 is the value which will be shifted out of the array, making the array arrName = [I2,...,IN+1]

Arithmetic operators

Arithmetic operators take numerical values literals or variables & return a single numerical value after computation.

Addition operator (+) :

Addition operator takes two numbers as operand & returns their computational sum as output. It can also be used for concatenation of two string (this is covered in the string section).

var x = 1;
var y = 2;
var z = x+ y;  //Output = 2
console. log( true+ true);  //Output = 2
console. log( true+ false);  //Output = 1
console. log( false+ false);  //Output = 0
console. log( true+ "Hello");  //Output = "1Hello" (string)

Subtraction operator (-) :

Subtraction operator takes two numbers as operand & returns their computational difference as output.

var x = 1;
var y = 2;
var z = x- y;  //Output = -1
console. log( 10- "Hello");  //Output = NaN (not-a-number)

Multiplication operator (*) :

Multiplication operator takes two numbers as operand & returns their computational multiplication as output.

var x = 1;
var y = 2;
var z = x* y;  //Output = 2
console. log( Infinity* Infinity);  //Output = Infinity
console. log( Infinity* -Infinity);  //Output = -Infinity
console. log( Infinity* 0);  //Output = NaN (not-a-number)
console. log( 2* "STRING");  //Output = NaN (not-a-number)

Division operator (/) :

Division operator takes two numbers as operand & returns their computational division as output.

var x = 1;
var y = 2;
var z = x/ y;  //Output = 0.5
z = x/ 0;  //Output = Infinity
z = x/ -0;  //Output = -Infinity

Basic arithmetic operators on floats :

Basic arithmetic operators (+,-,*,%) work exactly on floating point numbers as they work integers (as seen above).

var x = 1.2;
var y = 2.4;
var z = x+ y;  //Output = 3.6
z = x- y;  //Output = -1.2
z = x* y;  //Output = 2.88
z = x/ y;  //Output = 0.5

Increment operator (++) :

Increment operator increases the value of the variable by 1

var x = 1;
x++;  //Output = 2 (Is equivalent to x=x+1)

But, there are two types of increment operators :

  1. ++x (Prefix increment) : It increases the value of x, before returning the value.
  2. x++ (Postfix increment) : It increases the value of x, after returning the value.
Example of prefix & postfix increment :
var x = 10;
console. log( ++x);  //Output = 11, Value returned after printing = 11
console. log( x++);  //Output = 11, Value returned after printing = 12

Decrement operator (--) :

Decrement operator decreases the value of the variable by 1

var x = 1;
--x;  //Output = 0 (Is equivalent to x=x-1)

Once again, just like increment operators, there are two types of decrement operators :

  1. --x (Prefix decrement) : It decreases the value of x, before returning the value.
  2. x-- (Postfix decrement): It decreases the value of x, after returning the value.

Remainder operator (%) :

Remainder operator returns the remainder of division of two numbers. It is not the same as modulo function, because it doesn't returns the correct value of (-p)mod(m), where -p < 0 & m > 0. It can also be used to check, whether an integer is even or odd (e.g. 5%2=1, 4%2=0).

var x = 5;
x% 4;  //Output = 1

Compound assignment with augmented opration (+=, -=, *=, /=) :

var x = 10;
x+= 10;  //Output = 20 | x+=10 is same as x=x+10
x-= 10;  //Output = 10 | x-=10 is same as x=x-10
x*= 2;  //Output = 20 | x*=10 is same as x=x*10
x/= 10;  //Output = 2 | x/=10 is same as x=x/10
